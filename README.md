#Chat Easy Input tools.
These tools have been designed to *mostly* mimic the main site(s) input. Some variations exist and may possibly be changed to completly shadow the site. 

`alt` + `a`

* insert link

`alt` + `t`

* insert tag

`alt` + `b`

* bold text

`alt` + `i`

* italics text

`alt` + `s`

* strikethough text
